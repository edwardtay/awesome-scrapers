# Contributing to Awesome Scrapers

Thanks for your interest in contributing!

## Guidelines

### Adding a Tool

- **Actually useful**: Only add tools you've used or thoroughly evaluated.
- **Format**: Follow the existing table format with tool name (linked), stars, language, and a concise description.
- **Active**: Tools should have a commit within the last 12 months. If not, note it.
- **No duplicates**: Check the list first.
- **Right section**: Place the tool in the most relevant category.

### What belongs here

- Open-source scraping tools, frameworks, and libraries
- Managed scraping services and APIs (clearly labeled as commercial)
- Browser automation tools used for scraping
- Anti-detection and stealth tools
- Supporting tools (proxies, CAPTCHA solvers, parsers)

### What doesn't belong

- General-purpose HTTP libraries that aren't scraping-focused
- Tools with fewer than 100 GitHub stars (unless uniquely valuable)
- Personal scripts or one-off projects
- Tools that only exist to promote a single paid service

### Moving tools to the Graveyard

If a tool is unmaintained (no commits in 2+ years) or officially archived, submit a PR to move it to the Deprecated Tools Graveyard section with a recommended alternative.

## How to Submit

1. Fork the repo
2. Create a branch (`git checkout -b add-tool-name`)
3. Make your changes
4. Submit a PR with a brief explanation of why the tool belongs

## Star Counts

Star counts are approximate and updated periodically. Use `k` notation (e.g., `15k`) for readability.
